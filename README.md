# Easysoft Homebrew Tap

This is the [Homebrew][homebrew] [tap](https://docs.brew.sh/Taps.html) for
[Easysoft][easysoft].

[homebrew]: https://brew.sh/
[easysoft]: https://github.com/easysoft/homebrew-tap

## Usage

To install a formula from this tap, run:

```sh
brew tap easysoft/tap
brew install easysoft/tap/<formula>
```

## Formulae

| Formula | Description |
| ------- | ----------- |
| [qcadmin](https://github.com/easysoft/quickon_cli#%E4%BD%BF%E7%94%A8) | quickon cli |
